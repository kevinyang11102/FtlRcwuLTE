## 技术栈

- 前端技术：Vue、Element-UI
- 后端技术：Spring Boot、MyBatis Plus
- 数据库：MySQL
- 开发语言：Java

## 功能模块

### 系统角色

系统定义了两个主要角色：管理员和用户。

#### 管理员

- **登录功能**
- **用户管理**
  - 管理用户账户
- **订单管理**
  - 处理酒店预约订单
  - 查看订单列表
- **房间管理**
  - 房间列表展示
  - 房间详情查询
  - 添加房间信息
  - 修改房间信息
- **评论列表**
  - 查看用户评论

#### 用户

- **注册与登录**
- **查询空余房间**
- **房间预约**
- **个人中心**
  - 查看预约订单
